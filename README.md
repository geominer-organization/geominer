# GeoMiner: Sistema de Exploración Geológica con ML

GeoMiner es una plataforma digital diseñada para optimizar la planificación de perforaciones en la exploración minera. El proyecto busca reducir los altos costos operativos y el impacto ambiental de la minería tradicional, aplicando modelos predictivos de Machine Learning para identificar con mayor precisión las zonas de alto potencial mineral.

Este proyecto es desarrollado para el curso de IHC y Tecnologías Móviles.

## Segmentos Objetivo

La solución digital está diseñada para dos perfiles de usuario clave dentro de la industria minera:

* **Ingeniero de Planificación Minera:** Profesional encargado de diseñar y optimizar las campañas de perforación. Busca reducir costos, aumentar la precisión y validar la viabilidad técnica de los proyectos.
* **Ingeniero de Monitoreo Ambiental:** Profesional responsable de asegurar el cumplimiento normativo y minimizar la huella ecológica. Busca evaluar el impacto de las operaciones y seleccionar alternativas sostenibles.

## Características Principales

La solución GeoMiner se compone de una **Landing Page** y una **Aplicación Móvil**.

### Aplicación Móvil

* **Autenticación y Gestión de Proyectos:** Sistema seguro de inicio de sesión, registro y selección de proyectos mineros.
* **Visualización Geoespacial 2D:** Mapas interactivos separados para la vista Minera y la vista Ambiental, que muestran la ubicación de yacimientos, datos geofísicos y zonas sensibles.
* **Consulta de Datos:** Acceso a información detallada, histórica y reportes de cada yacimiento.
* **Búsqueda y Filtros:** Herramientas para filtrar yacimientos por tipo, ubicación u otros criterios en ambos mapas.
* **Mensajería Interna:** Un módulo de comunicación para que los equipos colaboren, gestionen contactos y compartan información técnica.
* **Gestión de Perfil:** Funcionalidades para que el usuario visualice su perfil, reciba notificaciones y cierre sesión.

### Landing Page

* **Plataforma Informativa:** Presenta la propuesta de valor del producto y la información corporativa.
* **Captación de Clientes:** Un formulario de contacto para que los visitantes soliciten información o una demostración.
* **Marketing de Contenidos:** Secciones para noticias y enlaces a perfiles sociales para mantener informada a la comunidad.
* **Acceso a la Plataforma:** Enlaces de Registro e Inicio de Sesión que conectan con el ecosistema de la aplicación móvil.
* **Diseño Adaptativo:** Interfaz completamente adaptativa para una correcta visualización en computadoras, tablets y celulares.
